# Yodo1-MAS-SDK-iOS-Demo

## Overview

MAS is Yodo1's in-app monetization solution.

Please check out our [documentation](https://developers.yodo1.com/article-categories/ios/) to get started on integrating.

## Demo App

The MAS SDK provides two demos for Cocoapods integration and Manual integration

* See [cocoapods/Yodo1MasSdkDemo](./cocoapods/README.md) for Cocoapods integration
* See [manual/Yodo1MasSdkDemo](./manual/README.md) for Manual integration
